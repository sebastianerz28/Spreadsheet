Info for Spreadsheet:
Files Must have the .sprd to be opened and saved
To open, save, and create new spreadsheets buttons are found in file
To select a cell you must left click
Selecting a cell updates the labels with the value and name of cell
To edit the contents of the cell you must type in the text box and
to update it you must hit the button to change contents of cell
Cells must be typed in as a single lowercase or uppercase letter followed by
the range of integers 1-99


Additional Features
Each button in file has a short cut
Red X button is overwritten to prompt you to save before you close your work
